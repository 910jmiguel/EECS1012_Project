Help


How to Navigate

Use the following navigation bar on the left of the screen. This will allow you to navigate to the home, profile (not available), course list, to-do list and the credits page.


Google Calendar API

There is a Google Calendar shown in the home page of the screen. This allows to display the current date and upcoming events. Note that there may be issues with the Google sign-in or reCaptcha


How to use the Weather Dashboard

Enter any city that is valid and it will display the current temperature, weather condition, and the hourly forecast


How to use the To-Do List

Add any task and you have the ability to remove it or complete it from the list


Additional Notes

Please note that the Course list does not work and is only used for display. The profile screen is also not displayed.
